# Website Performance Analysis

## Project Overview
This project analyzes website traffic and user engagement using Google Analytics data.

## Objectives
- Analyze sessions and users over time
- Identify top-performing traffic channels
- Compare engagement rates across channels
- Evaluate engaged vs non-engaged sessions
- Analyze hourly traffic patterns
- Study the relationship between sessions and engagement

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Google Analytics website performance dataset.

## Key Insights
- Identified the channels generating the highest traffic.
- Measured engagement performance across channels.
- Discovered peak traffic hours.
- Compared engagement trends with session volume.
